6、值得注意的点

运行参数：–name 可以标注容器名称，方便以后区分和删除。

命令：docker ps -a，加上 -a 参数可以列出所有容器（默认只会列出所有在运行的容器）。
主要作用是在docker停止或者非法退出（停电等）情况下，重启启动容器而不丢失记录。
