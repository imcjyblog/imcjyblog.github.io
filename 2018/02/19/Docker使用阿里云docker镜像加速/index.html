Docker使用阿里云docker镜像加速

发表于 2018-02-19 | 分类于 技术

先放出阿里云官网地址。

登录后，左边选择镜像加速器。

复制您的专属加速器地址，下方选择操作文档，针对不同系统进行配置。

我的macOS，因为装的是docker for mac，所以直接进配置，配置deamon，将地址加到”registry-mirrors”的数组里。

最后点击Apply & Restart按钮就完成了。
